🔵 Jira Integration

Connect your Jira Cloud workspace. Once configured, the tracker will show live ticket status, summary, and assignee when you open any entry with a Jira link.

Get your API token from id.atlassian.com → Security → API tokens. Token is stored locally and in Firebase — shared with your team.

🗂 Master Registry (per project)

The canonical list of masters for this project — each linked to its Jira issue key. Used to reliably connect testing, signoff, mapping and live Jira. Auto-generate from existing data + the epic, then fix any Jira keys.

🧪 Testing Sessions

The testing rounds for this project (Testing 1, Testing 2, …). Add as many as you need — they appear in the Testing Data filters and the Add Entry form.

🔗 Read-only Share Link

A view-only link to this project — recipients see all tabs but cannot edit. No sign-in required.

🗑 Data Management

Reset all testing data back to the original seed data. This cannot be undone.
